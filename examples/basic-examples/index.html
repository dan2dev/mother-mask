mother-mask

Lightweight input mask library. Pick an example below.

01 process() Pure function — mask a string without touching the DOM. 02 bind() Attach a live mask to an HTML input element. 03 Array masks Dynamic patterns that adapt to the input length. 04 bind() + callback React to every change and extract the raw value. 05 buildMask() / Mask Low-level class with caret tracking for custom editors. 06 Complete form Real-world registration form with live validation.
